# Simple Color detection repository

Python code for the implementation of the color detection.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install cv2, pandas and numpy.

```bash
pip install numpy
pip install cv2
pip install pandas
```

## Usage

```python
python interface.py
python tool.py
python main2.py
```
